title: Create New Monitor
description: Use the API to create a new monitor in your workspace.

## Required Permissions

-   `monitors.create`

## Return Values

The API will return a JSON response that can include the following values:

-   `error`: A boolean that indicates whether the request was successful or not.
-   `message`: If error is true, this will contain a message describing the error.
-   `monitor`: The monitor object that was created. You cna find their values in the table below.
